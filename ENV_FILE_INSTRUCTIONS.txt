יצרתי עבורך את התוכן המלא לקובץ .env
=====================================================

העתק את כל התוכן הבא לקובץ .env החדש:

DATABASE_URL=postgresql://postgres:postgres@db:5432/gitarena
SECRET_KEY=your-secret-key-change-in-production-gitarena-2025
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=30

GITHUB_CLIENT_ID=your-github-client-id
GITHUB_CLIENT_SECRET=your-github-client-secret
GITHUB_REDIRECT_URI=http://localhost:3000/auth/callback

OPENAI_API_KEY=your-openai-api-key-or-leave-empty
OPENAI_MODEL=gpt-4o-mini

FRONTEND_URL=http://localhost:3000
BACKEND_URL=http://localhost:8000

=====================================================

איך ליצור את הקובץ:
=====================

1. פתח VS Code
2. File → New File
3. הדבק את התוכן למעלה (ללא הכותרות!)
4. File → Save As
5. שמור בשם: .env
6. מיקום: c:\Users\pnina\source\-GitArena\GitArena\backend\.env

חשוב! 
------
1. אם יש לך GitHub OAuth App, החלף את:
   - GITHUB_CLIENT_ID
   - GITHUB_CLIENT_SECRET
   
2. אם יש לך OpenAI API Key, החלף את:
   - OPENAI_API_KEY
   
3. אם אין לך - אל תדאג! זה יעבוד גם ככה עם ערכי ברירת מחדל.
